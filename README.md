# number-guessing-game
A Python console game where the player tries to guess a randomly generated number between 1 and 100.

## Features

- Random number generation
- Unlimited guesses
- Input validation
- Handles non-numeric input
- Prevents out-of-range guesses
- Counts the number of attempts
- Gives feedback based on performance

## Concepts Used

- Variables
- Loops
- Conditional statements
- Exception handling (`try`/`except`)
- Random module
